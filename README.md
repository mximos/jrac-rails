# Jrac::Rails

ruby gem for popular jquery plugin jrac http://www.trepmag.ch/z/jrac/example/

## Installation

Add this line to your application's Gemfile:

    gem 'jrac-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jrac-rails
